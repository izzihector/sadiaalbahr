Stock Ageing Analysis

The stock ageing analysis is the process of monitoring the stock in our location based on their incoming date.With this module you can know the exact valuation of each product hold at current date.It will generate full report as per analysis criteria either locationwise or productwise or product category wise.You can get the detailed report about how older is your stock.With this module you can get the clear picture for each product that how much percentage of the product hold is in proportion to whole stock.

Features

  • Aging Quantity with Stock Value.
  • Stock Ageing Analysis with dynamic intervals.
  • Filters based on Products or Product categories, Stock locations.

Installation

After the successful installation of 'Stock Ageing Analysis' module new menu is available in Inventory.


Stock Ageing Analysis Wizard

Interval(Days): Dynamic Interval, Suppose if we enter interval days as 30 then system will create interval as 0-30, 30-60, 60-90, 90-120, 120+ in pdf and excel report.
Date: Select the date from which you want to see stock ageing analysis.
Filter By: You can select the option either you can filter by product or product category.
Location: You can select the locations of the company if Not selected then it will take products from all the locations.
Product: You can select the products here if not selected then it will take all the Products.
Product Category: You can select the product category here if Not selected then it will take products from all the Category.
